# Lab Tools

A premium collection of biomedical research calculators.

## Included Tools
1. **Dilution Calculator**: Solvent/Stock volume calculations ($C_1V_1 = C_2V_2$).
2. **Molarity Calculator**: Mass, Volume, Concentration conversions.
3. **BCA Protein Calculator**: Standard curve linear regression and sample analysis.

## How to Run Locally
Simply open the `index.html` file in any modern web browser.

## How to Share / Deploy
### Option 1: GitHub Pages (Recommended)
1. Create a new repository on GitHub.
2. Push these files to the repository.
3. Go to Settings > Pages and enable it for the `main` branch.

### Option 2: Netlify Drop
1. Drag and drop the `lab_tools` folder onto [Netlify Drop](https://app.netlify.com/drop).
2. It will generate a live link instantly.

### Option 3: Zip and Share
Compress the `lab_tools` folder and email/Slack it to your lab members. They just need to unzip and open `index.html`.
